# Directory Reference Map
> Path: `/references/directory-map.md`

| Directory | Purpose |
| :--- | :--- |
| `/src` | Application entrypoint and top-level routing/theme orchestration. |
| `/src/components` | Shared UI component library grouped by type. |
| `/src/components/charts` | Charting components for analytics dashboards. |
| `/src/components/common` | Navigation and header components used across pages. |
| `/src/components/layout` | Invoice layout templates for screen and PDF export. |
| `/src/components/other` | Utility widgets like KPI stat cards. |
| `/src/pages` | Route-level business workflows (orders, products, customers, etc.). |
| `/src/scenes` | Legacy dashboard/demo scene templates. |
| `/src/scenes/bar` | Bar chart demo scene. |
| `/src/scenes/calendar` | Calendar demo scene. |
| `/src/scenes/contacts` | Contacts data grid demo. |
| `/src/scenes/dashboard` | Full legacy dashboard scene. |
| `/src/scenes/faq` | FAQ layout demo. |
| `/src/scenes/form` | Form layout demo. |
| `/src/scenes/geography` | Geographic chart demo. |
| `/src/scenes/global` | Legacy scene sidebar/topbar components. |
| `/src/scenes/invoices` | Invoice list demo scene. |
| `/src/scenes/line` | Line chart demo scene. |
| `/src/scenes/pie` | Pie chart demo scene. |
| `/src/scenes/team` | Team list demo scene. |
| `/src/server` | Supabase integration, database scripts, and schema assets. |
| `/src/server/supabase` | Supabase client setup and reusable data helpers. |
| `/src/server/scripts` | Database maintenance and migration scripts. |
| `/src/server/database` | SQL schema files and RLS policies. |
| `/src/server/database/mocks` | Mock datasets for chart demos and development. |
| `/src/utils` | Security utilities and governance guidance. |
