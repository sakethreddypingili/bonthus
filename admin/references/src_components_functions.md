# Modular Functional Reference: src/components

This document provides an alphabetical list of all explicitly named components and exported utilities within the `src/components` directory tree.

## Components & Functions

### B
- **BarChart** (`src/components/charts/BarChart.jsx`): Functional component rendering a responsive Nivo bar chart.
- **BoardSidebar** (`src/components/common/BoardSidebar.jsx`): Functional component providing navigation for the Board module.

### G
- **GeographyChart** (`src/components/charts/GeographyChart.jsx`): Functional component rendering a responsive Nivo choropleth map.

### H
- **Header** (`src/components/common/Header.jsx`): Functional component for standard page headers (title/subtitle).

### I
- **InvoiceLayout** (`src/components/layout/InvoiceLayout.jsx`): ForwardRef functional component for on-screen invoice rendering.

### L
- **LineChart** (`src/components/charts/LineChart.jsx`): Functional component rendering a responsive Nivo line chart.

### P
- **PdfInvoiceLayout** (`src/components/layout/PdfInvoiceLayout.jsx`): ForwardRef functional component for print-optimized invoice rendering.
- **PieChart** (`src/components/charts/PieChart.jsx`): Functional component rendering a responsive Nivo pie chart.
- **ProgressCircle** (`src/components/charts/ProgressCircle.jsx`): Functional component for a radial progress indicator.

### S
- **Sidebar** (`src/components/common/Sidebar.jsx`): Functional component for the main application sidebar.
- **StatBox** (`src/components/other/StatBox.jsx`): Functional component for displaying a KPI card with an icon and progress circle.

### T
- **Topbar** (`src/components/common/Topbar.jsx`): Functional component for the global top navigation bar.
