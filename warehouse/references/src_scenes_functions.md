# Modular Functional Reference: src/scenes
> Path: `/src/scenes`

## Components & Functions

- `Bar` (Component) - Renders the Bar chart scene.
- `Calendar` (Component) - Renders the interactive Calendar scene.
- `Contacts` (Component) - Renders the Contacts data grid scene.
- `Dashboard` (Component) - Renders the main legacy Dashboard scene with multiple widgets.
- `FAQ` (Component) - Renders the FAQ accordion scene.
- `Form` (Component) - Renders the user creation form scene.
- `Geography` (Component) - Renders the Geography choropleth map scene.
- `handleDateClick` (Function) - Handles date selection in the Calendar component to add new events.
- `handleEventClick` (Function) - Handles event clicks in the Calendar component to delete events.
- `handleFormSubmit` (Function) - Processes form submission in the Form component.
- `Invoices` (Component) - Renders the Invoices data grid scene.
- `Item` (Component) - Helper component for rendering individual menu items in the Sidebar.
- `Line` (Component) - Renders the Line chart scene.
- `Pie` (Component) - Renders the Pie chart scene.
- `Sidebar` (Component) - Renders the legacy navigation sidebar.
- `Team` (Component) - Renders the Team management data grid scene.
- `Topbar` (Component) - Renders the legacy navigation topbar.
